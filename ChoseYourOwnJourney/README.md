A simple text based adventure game where the player makes choices, collects items, and encounters random enemies. Perfect for beginners learning Python conditionals, input handling, and basic game logic.

Features
Branching story paths

Inventory system

Random enemy encounters

Beginner friendly code structure

Pull requests are welcome. Feel free to add:

new story branches

more items

combat system

save/load features

Code Structure:

adventure.py
Player name input
First choice (climb wall / hidden path)
Second choice (item collection)
Inventory system
Random enemy encounter
