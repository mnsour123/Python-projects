A simple text based adventure game where the player makes choices, collects items, manages health, and encounters random enemies. Perfect for beginners learning Python conditionals, input handling, and basic game logic.

## Features

- Branching story paths (with deeper sub-branches to explore)
- Inventory system
- Health system
- Item-based combat outcomes (some items help you fight or escape better than others)
- Random enemy encounters
- Multiple endings based on your choices and inventory
- Beginner friendly code structure

## Pull requests are welcome. Feel free to add:

- new story branches
- more items
- combat system improvements
- save/load features
- difficulty levels
- ASCII art / visuals

## Code Structure

`choseyourownjourney.py`
- Player name input
- First choice (climb wall / hidden path)
- Second choice (item collection)
- Sub-branches for deeper exploration (e.g. explore the tower, follow the map)
- Inventory system
- Health tracking
- Random enemy encounter
- Combat logic based on inventory items
- Ending logic based on final health and inventory
