# Pig Dice Game

A simple command-line dice game for 24 players, written in Python.

## Rules

- Roll the die on your turn to add to your turn score.
- Roll a **1** and you lose all points from that turn.
- Choose to stop anytime to bank your turn score.
- First to **70 points** wins.

## Run

```bash
python pig_game.py
```

## Requirements

Python 3.x, no dependencies.
