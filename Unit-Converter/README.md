# Unit Converter

A simple GUI unit converter built with Python's `tkinter`.

## Features

- Convert between **Length**, **Weight**, and **Temperature**
- Auto-updating unit dropdowns based on selected category
- Clean, lightweight interface (no external dependencies)

## Supported Units

- **Length:** Meters, Kilometers, Miles, Feet
- **Weight:** Kilograms, Pounds, Grams, Ounces
- **Temperature:** Celsius, Fahrenheit, Kelvin

## Requirements

- Python 3.x (tkinter comes built-in with standard Python installs)

## How to Run

```bash
python unit_converter.py
```

## Usage

1. Select a category (Length, Weight, or Temperature)
2. Enter a value
3. Choose the "from" and "to" units
4. Click **Convert** to see the result
